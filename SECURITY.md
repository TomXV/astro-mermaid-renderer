# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 0.1.x | ✅ |

## Reporting a Vulnerability

Please **do not** report security vulnerabilities through public GitHub Issues.

Instead, report them via [GitHub's private vulnerability reporting](https://github.com/TomXV/astro-mermaid-renderer/security/advisories/new).

You can expect:
- Acknowledgement within 3 business days
- A fix or mitigation plan within 14 days for confirmed vulnerabilities

Thank you for helping keep this project secure.
